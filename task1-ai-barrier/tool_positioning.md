# 工具定位

## AI 工具生态定位分析

### 工具分类

| 类别 | 代表工具 | 核心定位 | 适用场景 |
|------|---------|---------|---------|
| 通用对话 | ChatGPT, Claude | 广泛知识问答、文本生成 | 日常咨询、写作辅助 |
| 代码助手 | GitHub Copilot, Claude Code | 代码补全、重构、调试 | 软件开发全流程 |
| 图像生成 | Midjourney, DALL-E | 创意视觉内容生成 | 设计原型、概念图 |
| 数据分析 | ChatGPT Code Interpreter | 数据处理与可视化 | 报表分析、数据洞察 |
| 搜索增强 | Perplexity, Bing Chat | 实时信息检索+总结 | 研究调研、事实核查 |

### 我的工具矩阵

- **主力工具**: Claude Code — 代码开发、项目管理
- **辅助工具**: ChatGPT — 快速问答、头脑风暴
- **学习工具**: GitHub Copilot — 代码补全、模式学习

### 工具协作策略

1. 复杂任务 → Claude Code（深度推理 + 工具链）
2. 快速验证 → ChatGPT（响应快、覆盖面广）
3. 代码实现 → Copilot + Claude Code 互补
